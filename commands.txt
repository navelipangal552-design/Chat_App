npm i--> short for npm install

express@4.21.2 -->

Web framework for Node.js
Simplifies building web servers and REST APIs
Provides easy routing, middleware, and request handling

mongoose@8.10.1 -->

MongoDB Object Data Modeling (ODM) library
Connects your Node.js app to MongoDB
Lets you define schemas and models

jsonwebtoken@9.0.2 --> Autentication

Used for JWT (JSON Web Token) authentication
Helps generate and verify tokens for user login systems
Commonly used to secure APIs and manage sessions

bcryptjs@2.4.3 --> Security

Library for hashing passwords
Encrypts passwords before storing them in your database
Protects user data even if the DB is compromised

dotenv@16.4.7 -->

Loads environment variables from a .env file into process.env

cookie-parser@1.4.7 --> 

Middleware that parses cookies from incoming HTTP requests.
When users log in, servers often send a cookie to store tokens or session info.
cookie-parser lets Express easily read those cookies from requests.
